# Hoe te gebruiken

1. Download de `app.py` en zet deze in een mapje.
2. Maak een mapje genaamd `data` in hetzelfde mapje als de `app.py`.
3. Zet in de `data` map het document met de leidingsvorming data.
4. Uw werkmap heeft normaal gezien de volgende structuur: 
    ```
    LeidingsvormingAutomatisatie/
    ├── app.py
    └── data/
        └── Leidingsvorming.xlsx
    ```
5. Start de `app.py`
